# Meetup Pet-Friendly Places

¡Bienvenido a la landing page de Meetup para lugares aptos para mascotas!

Este proyecto es un prototipo de página de aterrizaje para un servicio que ayuda a dueños de mascotas a descubrir, compartir y unirse a meetups en lugares pet-friendly.

## Características
- Hero visual con llamada a la acción
- Formulario para registro de correo (early access)
- Imagen representativa de mascotas
- Sección "How It Works" con pasos e íconos
- Diseño moderno y responsivo

## Archivos principales
- `index.html`: Estructura principal de la página
- `style.css`: Estilos visuales
- `script.js`: Lógica del formulario

## Cómo usar
1. Clona el repositorio:
   ```sh
   git clone https://github.com/AlfredoZarateSFTK/Mascotas.git
   ```
2. Abre el archivo `index.html` en tu navegador web.

## Personalización
Puedes cambiar imágenes, textos y colores editando los archivos mencionados.
